# DispatchBenchmark

Benchmark project for Swift method dispatch.

## Requirement

- Xcode 10.0~
- Swift 4.2

## Benchmark

```sh
$ swift run

| static  | 508            | 8169            | 48134            | 495011            |
| vtable  | 1072           | 7982            | 52529            | 563639            |

```
